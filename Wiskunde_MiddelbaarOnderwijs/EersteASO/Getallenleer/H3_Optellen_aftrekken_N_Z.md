# Optellen en aftrekken in N en Z

## De optelling

Rekenregel: getallen met hetzelfde toestandsteken

- behoud het toestandsteken
- tel de absolute waarden op

Rekenregel: getallen met verschillend toestandsteken

- neem het toestandsteken van het getal met de grootste absolute waarde
- trek de kleinste van de grootste absolute waarde af

bv. -4 + 3 = -1

### Eigenschappen

#### Overal gedefinieerd

De som van twee natuurlijke getallen is altijd een natuurlijk getal

De som van twee gehele getallen is altijd een geheel getal

#### Commutatief

Bij de optelling van natuurlijke getallen met je de termen van plaats verwisselen, de som blijft hetzelfde

a + b = b + a

Bij de optelling van natuurlijke getallen met je de termen van plaats verwisselen, de som blijft hetzelfde

#### Associatief

Bij het optellen van natuurlijke getallen mag je haken verplaatsen, toevoegen of weglaten, de som blijft hetzelfde

(a + b) + c = a + (b + c) = a + b + c

Bij het optellen van gehele getallen mag je haken verplaatsen, toevoegen of weglaten, de som blijft hetzelfde

#### Neutraal element

0 is het neutraal element. De som van 0 en een natuurlijk getal is altijd een natuurlijk getal. De som van 0 en een geheel getal is altijd een geheel getal

a + 0 = a = 0 + a

#### Symmetrisch element

Als je bij een geheel getal zijn tegengestelde optelt, bekom je het neutraal element voor de optelling

a + (-a) = 0 = (-a) + a

#### Gedurige som

= de som van meer dan twee termen

## De aftrekking

### Reken- en tekenregels

Aftrekken is inverse bewerking van het optellen.

Om twee gehele getallen af te trekken, tel bij het eerste getal het tegengestelde van het tweede getal op

bv. a - b = a + (-b)

### Vereenvoudigde schrijfwijze

Twee dezelfde tekens > +

Twee verschillende tekens > -

### Eigenschappen van het aftrekken in N en Z

#### Overal gedefinieerd bij aftrekken

Aftrekking van twee natuurlijke getallen is niet altijd een natuurlijk getal. Aftrekken in N is niet overal gedefinieerd.

Aftrekking van twee gehele getallen is altijd een geheel getal. Aftrekken in Z is overal gedefinieerd.

#### Commutatief bij aftrekken

Bij natuurlijke en gehele getallen mag je de termen NIET van plaats verwisselen. Het verschil is niet hetzelfde. De aftrekking is niet commutatief.

#### Associatief bij aftrekken

Bij natuurlijke en gehele getallen mag je de haakjes NIET verplaatsen, weglaten of toevoegen. De aftrekking is niet associatief.

#### Neutraal element bij aftrekken

De aftrekking in N en Z heeft geen neutraal element.

#### Symmetrisch element bij aftrekken

Aangezien er geen neutraal element is, is er ook geen symmetrisch element.

### Hakenregel

Plusteken voor de haken mag je de haken weglaten.

Minteken voor de haken, dan mag je de haken weglaten en elke term binnen de haken van teken veranderen.

bv -(a-b) = -a + b
