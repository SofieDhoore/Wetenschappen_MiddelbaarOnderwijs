# Soorten getallen

## De natuurlijke getallen

Een natuurlijk getal is een telresultaat: 0, 1, 2,...

## De gehele getallen

Negatieve én positieve getallen: -2, -1, 0, 1, 2,...

**Absolute waarde**: de absolute waarde van een geheel getal is dat getal zonder toestandsteken

|a| = de absolute waarde van a

**Tegengestelde getallen**: gehele getallen met eenzelfde absolute waarde maar een verschillend toestandsteken

-(a) = het tegengestelde van a

## De rationale getallen

Een rationaal getal is het quotiënt van twee gehele getallen waarvan de deler niet 0 is
